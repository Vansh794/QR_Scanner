QR Scanner Web App

A simple and responsive QR Code Scanner web application built using HTML, CSS, and JavaScript. This project allows users to scan QR codes directly using their device camera and process/store the scanned data.

🚀 Features
📷 Scan QR codes using device camera
⚡ Fast and lightweight interface
💻 Responsive design for desktop and mobile
🔒 Browser-based scanning (no app installation required)
🗄️ Ready for database integration (PostgreSQL support can be added)
🛠️ Tech Stack
HTML5
CSS3
JavaScript
QR Scanner Library
📂 Project Structure
QR_Scanner/
│── index.html
│── style.css
│── script.js
│── assets/
└── README.md
📸 How It Works
Open the website
Click on the Scan QR button
Allow camera permissions
Point the camera toward a QR code
The scanned data will be displayed and can be stored in a database
⚙️ Installation & Setup

Clone the repository:

git clone https://github.com/Vansh794/QR_Scanner.git

Open the project folder:

cd QR_Scanner

Run the project by opening index.html in your browser.
